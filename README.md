# EEZ — Ethereum Economic Zone Technical One-Pager

A standalone HTML one-pager covering the full technical overview of the Ethereum Economic Zone, built in the EEZ brand design language.

## Preview

Open `eez-onepager.html` in any browser. No build step, no dependencies, no server required.

## Contents

```
eez-onepager.html   ← the full page
img/                ← 16 use case renders (8 use cases × problem + solved)
```

## Sections

1. **Hero** — title, tagline, founding partners
2. **Problem / Solution** — fragmentation state vs. unified EEZ zone
3. **Technical Architecture** — composability matrix, 5-step execution flow, smart contract structs
4. **Key Technical Specifications** — proving time, security, prover diversity, protocol changes, gas token, settlement
5. **Use Cases** — 8 image cards (hover any card to flip from solved → problem state)
6. **What EEZ Brings to Ethereum** — settlement dominance, economic realignment, network effects, credible neutrality, composability
